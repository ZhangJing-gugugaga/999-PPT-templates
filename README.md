## 使用说明

所有模板均为 `.pptx` 或 `.ppt` 格式，兼容 Microsoft PowerPoint、WPS Office、Google Slides 等主流演示软件。

### 模板目录

```
├── 学术风/     # 论文答辩、学术报告类模板
├── 党建风/     # 党政机关、红色主题模板
├── 中国风/     # 国潮、水墨等中国元素模板
├── 商务风/     # 商业计划、工作汇报模板
├── 手帐风/     # 手绘风格、日常记录模板
├── 图与表/     # 图表、分析数据可视化模板
├── 艺术风/     # 创意设计、艺术风格模板
├── 科技风/     # 科技互联网风格模板
└── 小彩蛋/     # 彩蛋模板
```
